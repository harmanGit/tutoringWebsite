&lt;head&gt; == used for the head of the html page
    &lt;head&gt;
        &lt;title&gt;First website&lt;/title&gt;
        &lt;meta charset="UTF-8"&gt;
    &lt;/head&gt;

&lt;body&gt; == used for the body of the html page
    &lt;body id="body"&gt; &lt;div&gt;welcome to my webpage &lt;/div&gt; &lt;/body&gt;

&lt;p&gt; == the paragraph tag, used for text. Adds a newline and the beginning and the end
    &lt;p&gt;Imma paragraph&lt;/p&gt;

&lt;strong&gt; == bolds text
    &lt;strong&gt;Is this text bold???&lt;/strong&gt;

&lt;i&gt; == Italic's text
    &lt;i&gt;Is this text italic&lt;/i&gt;

&lt;h1&gt; -&gt; &lt;h6&gt; == Heading tag, H1 is the biggest and it goes down in size...H2, H3, H4, H5, H6
    &lt;h1&gt;Big HEADING text yayyy!&lt;/h1&gt;

&lt;div&gt; == Dummy tag used to hold other attributes, adds a newline at the end
    &lt;div&gt;welcome to my webpage &lt;/div&gt;

&lt;span&gt; == Dummy tag used to hold other attributes, inline and no newlines anywhere
    &lt;span&gt;Hello im a inline span tag&lt;/span&gt;

&lt;a&gt; == the anchor tag, could be used to link to other pages
    &lt;a href="cssPage.html"&gt;CSS Start Guide&lt;/a&gt;

&lt;img&gt; == the image tag, used to place images into your webpage
    &lt;img src = “immaImage.jpeg” alt = “If you see this the image tag is broken” width = “100” height=”100” &gt;

&lt;ul&gt; == the unordered list tag, items in the list are bulleted
    &lt;ul&gt; Im a unordered list &lt;li&gt;item&lt;/li&gt;&lt;/ul&gt;

&lt;ol&gt; == the ordered list tag, items in the list are numbered
    &lt;ol&gt; Im a ordered list &lt;li&gt;item&lt;/li&gt;&lt;/ol&gt;

&lt;li&gt; == defines a item in the list
    &lt;li&gt;item in any of the two lists above&lt;/li&gt;

&lt;dl&gt; == the definition tag, each item has another item in front of it
&lt;dt&gt; == definition of the item
&lt;dd&gt; == defines a item for the definition
    &lt;dl&gt;&lt;dt&gt;Definition Name&lt;/dt&gt;&lt;dd&gt;Item that defines the definition&lt;/dd&gt;&lt;/dl&gt;
